Descripción detallada

T.D.A. Sopa de Letras.

Una instancia c del tipo de datos abstracto Sopa de Letras contiene un conjunto de palabras dispuesto en la direcciones verticales, horizontal o diagonal en una matriz dispersa. Se representa:

[{fila1,columna1,letra},...,{filaN,columnaN,valorN}]
